# Clustering with PCA & DBSCAN

## Overview
This project applies PCA for dimensionality reduction and DBSCAN for clustering, and compares results before and after PCA.

## Steps
- Data preprocessing (cleaning and scaling)
- Apply PCA (2 components)
- Apply DBSCAN on original and PCA data
- Tune parameters (eps, min_samples)
- Visualize and analyze results
